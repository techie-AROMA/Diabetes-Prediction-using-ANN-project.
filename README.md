Diabetes Prediction System Using Artificial Neural Network (ANN)

	Introduction:

 Diabetes is one of the most common chronic diseases affecting millions of people worldwide. Early detection of diabetes can help prevent severe health complications such as heart disease, kidney failure, and vision problems. With the advancement of Artificial Intelligence and Machine Learning, predictive systems can be developed to assist medical professionals in identifying potential diabetic patients based on medical attributes. This project focuses on building a Diabetes Prediction System using an Artificial Neural Network (ANN). The model is trained on a dataset containing medical attributes such as glucose level, blood pressure, BMI, age, and insulin level. The trained model predicts whether a person is likely to have diabetes.

	Problem Statement:

 The increasing number of diabetes cases worldwide highlights the need for efficient prediction systems. Traditional diagnosis methods can be time consuming and require extensive medical testing. A machine learning based prediction model can assist doctors by providing quick predictions based on existing patient data. The main objective of this project is to build a deep learning model that can predict whether a patient has diabetes based on medical parameters.

	Dataset Description:

The dataset used in this project is the Pima Indians Diabetes Dataset. It contains medical information collected from female patients of Pima Indian heritage. The dataset consists of several medical predictor variables and one target variable. Features in the dataset include: - Pregnancies - Glucose - Blood Pressure - Skin Thickness - Insulin - BMI (Body Mass Index) - Diabetes Pedigree Function - Age The target variable indicates whether the patient has diabetes (1) or not (0).

	Methodology :

The methodology used in this project consists of the following steps: 1. Data Collection – Import the dataset. 2. Data Preprocessing – Handle missing values and scale the features. 3. Train Test Split – Divide the dataset into training and testing sets. 4. Model Building – Create an Artificial Neural Network model. 5. Model Training – Train the ANN model using training data. 6. Evaluation – Measure accuracy using test data. 7. Deployment – Build a simple web interface using Streamlit.

	ANN Model Architecture :

The Artificial Neural Network used in this project consists of: - Input Layer - Two Hidden Layers - Output Layer The hidden layers use the ReLU activation function to learn patterns from the data. The output layer uses the sigmoid activation function to perform binary classification.

	Tools and Technologies:

 Tools and Technologies Used: Programming Language: Python Libraries: - Pandas - NumPy - Scikit-learn - TensorFlow / Keras - Streamlit Development Environment: VS Code / Jupyter Notebook

	Results and Discussion:

 After training the model, the system was able to predict diabetes with good accuracy. The performance of the model depends on the quality of the dataset and the tuning of model parameters. The model was evaluated using metrics such as: - Accuracy - Confusion Matrix The system demonstrates how deep learning can be used to predict medical conditions


	 Conclusion:

 This project demonstrates the use of Artificial Neural Networks for predicting diabetes based on medical attributes. The developed model provides a simple yet effective way to predict the likelihood of diabetes. Future improvements may include: - Using larger datasets - Applying advanced deep learning architectures - Integrating the system with hospital databases - Building a mobile application for easier access
